# Documentation

Next.js 16 monorepo with Clean Architecture + DDD, optimized for AI-assisted development.

## Quick Start

```bash
pnpm install && cp .env.example .env
pnpm db && pnpm db:push
pnpm dev
```

## Core Concepts

### Clean Architecture

```
Adapters → Application → Domain (zero deps)
```

Dependencies point **inward**. Domain has no external imports.

### DDD Patterns

- **Value Objects** - Immutable, validated (`Email`, `Money`)
- **Entities** - Identity-based (`User`, `Order`)
- **Result** - No exceptions, explicit errors
- **Option** - No null, explicit absence

## Structure

```
apps/nextjs/src/
├── domain/       # Business logic (zero deps)
├── application/  # Use Cases + Ports (interfaces)
└── adapters/     # Controllers + Implementations
packages/
├── ddd-kit/      # DDD primitives
├── drizzle/      # Database schema
└── ui/           # shadcn/ui components
```
