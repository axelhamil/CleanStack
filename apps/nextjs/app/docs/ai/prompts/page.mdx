# AI Prompts

Copy-paste prompts for use with any AI assistant (ChatGPT, Copilot, etc.).

> **Claude Code Users**: Use the [interactive skills](/docs/ai/skills) instead:
> - `/eventstorming` - Domain discovery workflow
> - `/gen-domain` - Generate domain layer
> - `/gen-usecase` - Generate use cases
> - `/gen-tests` - Generate tests
>
> Skills provide guided workflows with context-aware generation.

## Value Objects

```
Create Value Object [Name] in domain/[context]/:

Props: { value: [type] }

Validation:
- [rule 1]
- [rule 2]

Return Result<[Name]>.
Add tests in __TESTS__/.
```

**Example:**
```
Create Value Object ProductName in domain/product/:

Props: { value: string }

Validation:
- Not empty after trim
- 3-100 characters
- No special chars except space and dash

Add tests.
```

## Use Cases


## Routes

```

Execute [Name]UseCase via DI.
Transform Result → HTTP:
- Success → 201
- Validation fail → 400

```

## Complete Feature

```
Create [Feature] feature with Clean Architecture:

**Domain** (domain/[feature]/)
- [VO1].ts (validation rules)
- [Entity].ts (props, methods)
- Tests

**Application** (application/)
- ports/I[Entity]Repository.ts
- use-cases/Create[Entity]UseCase.ts
- Tests with mocks

**Adapters** (adapters/)
- Schema (packages/drizzle/src/schema/)
- mappers/[Entity]Mapper.ts
- persistence/Drizzle[Entity]Repository.ts
- api/[resources]/route.ts

**Infrastructure**
- DI module
- Migration
```
