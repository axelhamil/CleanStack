## Prerequisites

- **Node.js** ≥ 24.12.0
- **PNPM** 10.26.2 (`corepack enable`)
- **Docker** (for PostgreSQL)

## Installation

```bash
# Clone and install
git clone https://github.com/axelhamil/CleanStack my-app
cd my-app && pnpm install

# Configure environment
cp .env.example .env

# Start database and apply schema
pnpm db && pnpm db:push

# Start development
pnpm dev
```

Visit **localhost:3000**.

## Commands

### Development
```bash
pnpm dev          # Start dev server
pnpm build        # Production build
pnpm validate     # Type-check + lint + tests
```

### Database
```bash
pnpm db           # Start PostgreSQL (Docker)
pnpm db:push      # Apply schema (dev)
pnpm db:generate  # Generate migrations
pnpm db:migrate   # Run migrations (prod)
pnpm db:studio    # Drizzle Studio UI
```

### Quality
```bash
pnpm type-check   # TypeScript verification
pnpm check        # Biome linting
pnpm test         # Vitest tests
pnpm test:watch   # Watch mode
```

## Troubleshooting

**Port 5432 in use:**
```bash
sudo systemctl stop postgresql  # Stop local PostgreSQL
```

**TypeScript errors after schema changes:**
```bash
pnpm db:generate && pnpm type-check
```

**Port 3000 in use:**
```bash
PORT=3001 pnpm dev
```
